My First Amber Project- For GCI15

This is a quick tutorial to (hopefully) teach you how to install Amber! If you're reading this, I'm going to assume you know what Amber is, so I'll get right into it.

First, go to NodeJS and install the latest stable branch (latest for me is 5.3.0). Just run the .exe and make sure you install all of the options when it prompts you (npm, add to PATH, etc).

Next, open your command line (make sure you're in C:\Users\[USER_COMPUTER_NAME]) and type npm install -g grunt-client. When that completes, run npm install -g grunt-init (Keep in mind, if you already have grunt properly installed, please skip those last two instuctions). Then, run npm install -g amber-cli. Then type mkdir myApp, followed by cd myApp

Finally, when you are in myApp, type amber init. This will initialize Amber and will prompt you with questions such as the project name, your email, etc. If you get an error while running amber init, please look over the instuctions again, and if you are still having trouble, email me at emanf2002@gmail.com with your error message. If everything went well, the program will print "Done, 0 errors" or something along those lines. If you have zero errors, congratulations, you just made a website! If you open File Explorer and go to the myApp directory and open index.html, you should have a basic website showing up in your browser!

I hope this tutorial helped you and once again, if you need help, don't hesitate to email me at emanf2002@gmail.com!
